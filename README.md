# Video Management Application

A full-stack application for recording, storing, and playing videos with a beautiful, responsive interface.

![Demo App](https://postimg.cc/6TnVfYHr)

## Tech Stack

- **Frontend:**
  - React
  - TypeScript
  - Tailwind CSS
  - Framer Motion
  - Axios

- **Backend:**
  - Node.js
  - Express
  - MongoDB
  - Mongoose
  - Multer
